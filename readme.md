# Prosty ekwipunek
Prosty ekwipunek działający na zasadzie *drag and drop* z dodatkową opcją łączenia wybranych przedmiotów (patyk + lina → wędka, patyk + grot → włóćznia).

**Wersja demonstracyjna :** http://deykun.boo.pl/lab/draganddrop/
